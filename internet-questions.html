About The Internet

The Internet of Things (IOT) is the giant network of devices connected to the internet. All of these devices collect and exchange data with one another. Some examples of things connected in the Internet of Things are smart phones, smart security systems, smart home appliances, smart speakers, and ai voice assistants.

Each time a browser loads a webpage, it loads and parses the HTML, which it recieves from the network. Next, it converts the HTML into a DOM (Document Object Model), which is the representation of the document in the computer's memory. After that process, the browser fetches all resources (except JavaScript) that are linked to the HTML document. The browser then parses the CSS and sorts out the different rules by their selector types and finds out which rules should be applied to which nodes in the DOM. Then the render tree is laid out and a visual display of the page is shown on the screen.

Cookies are small bits of data that are created by a web server while a user browses a website. Cookies are used to identify your device in order to store information about your prefrences and visit to the website. Usually cookies are collected when you press accept on the pop-up message that appears when you visit a new website. Cookies can improve user experience, but can also misuse private information.

The internet is a global network of billions of computers and other electronic devices. The internet works by using a packet routing network. Packets are small segments of data that gets sent over the internet. This packet routing network follows Internet Protocol (IP) and Transport Control Protocol (TCP). These are protocals for sending packets between devices. When you surf the internet, your device is connected to the web through a motem or router. When you type in a web address (URL) into your browser, your query is processed and pushed to your ISP (Internet Service Provider). The browser looks up the IP address for the domain name you typed into your browser through DNS (Domain Name Server). Then the browser sends a hypertext transfer protocol (HTTP) request to the server to send a copy of the web page to the client using TCP and IP.
